# HTML Text Formatting Visualizer

This project is an interactive webpage that demonstrates different **HTML text formatting tags** such as `<b>`, `<strong>`, `<i>`, `<em>`, `<mark>`, `<small>`, `<del>`, `<ins>`, `<sub>`, and `<sup>`.

The page presents each tag as an **interactive 3D card** where users can view the formatted text on the front and the explanation with code on the back.

The goal of this project is to learn and demonstrate how **HTML text formatting works along with modern UI interactions using CSS and JavaScript.**

---

## Features

- Interactive cards explaining HTML formatting tags
- Flip animation to view code and explanations
- Editable preview text
- Smooth scrolling gallery
- Hover tilt effect on cards
- Particle background animation
- Special modes like **X-Ray Mode** and **Matrix Mode**
- Responsive design for mobile devices
